# Beim Arzt

**Arzthelferin:** Guten Tag, haben Sie einen Termin?
**Patient:** Ja, mein Name ist Weber, ich habe um 10 Uhr einen Termin bei Frau Dr. Schneider.
**Arzthelferin:** Einen Moment bitte … Ja, Frau Dr. Schneider erwartet Sie. Sie können in Zimmer 3 gehen. Nehmen Sie bitte Platz, die Ärztin kommt gleich.
**Patient:** Danke. Ich habe seit drei Tagen starke Kopfschmerzen und Fieber. Gestern war ich bei 38,5 Grad.
**Arzthelferin:** Das klingt nach einer Grippe. Hatten Sie schon mal Corona? … Gut, dann warten Sie bitte, die Ärztin spricht mit Ihnen und macht vielleicht einen Schnelltest.
