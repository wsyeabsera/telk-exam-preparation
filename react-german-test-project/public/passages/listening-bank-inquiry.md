# In der Bank

Guten Tag. Ich möchte ein Girokonto eröffnen. – Haben Sie einen Termin? – Nein. – Einen Moment. … Sie können in einer halben Stunde dran sein. Brauchen Sie einen Reisepass oder Personalausweis. – Den habe ich dabei. – Gut. Für das Konto fallen monatlich vier Euro Gebühren an. Die Karte kommt per Post in etwa einer Woche. Online-Banking ist inklusive. Möchten Sie eine Kreditkarte dazu? – Nein, danke. Nur die EC-Karte. – Alles klar. Bitte füllen Sie das Formular aus und warten Sie, bis Sie aufgerufen werden.
