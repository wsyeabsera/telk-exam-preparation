# Aushang: Bibliothek

**Wichtige Mitteilung**

Vom 15. bis 22. Juni bleibt die Bibliothek wegen Renovierung **geschlossen**.

In dieser Zeit können keine Bücher ausgeliehen oder zurückgegeben werden. Fällige Rückgabetermine in dieser Woche werden automatisch verlängert. Keine Mahngebühren.

**Ab dem 23. Juni** wieder normale Öffnungszeiten: Mo–Fr 10–18 Uhr, Sa 10–14 Uhr.

Online-Katalog und Verlängerung unter www.stadtbibliothek.de.
