# Prepositions – Exercises 01

**Instructions / Anweisungen:** Fill in the correct form (Dativ or Akkusativ) after the preposition. Consider whether the preposition always takes one case or is two-way (wo? = Dativ, wohin? = Akkusativ). Answers at the bottom.

---

## Part 1: Regular Questions (1–15)

- [ ] 1. Sie fährt ___ (mit, der Bus).
- [ ] 2. Ich warte ___ (auf, du).
- [ ] 3. Er wohnt bei ___ (sein) ___ (alt-) Mutter.
- [ ] 4. Wir freuen uns ___ (auf, das Wochenende).
- [ ] 5. Sie kommt aus ___ (die) Türkei.
- [ ] 6. Nach ___ (das) Arbeit geht er nach Hause.
- [ ] 7. Ich denke ___ (an, du) oft.
- [ ] 8. Sie arbeitet seit ___ (ein) Jahr hier.
- [ ] 9. Er geht zu ___ (der) Arzt. (zum)
- [ ] 10. Ohne ___ (gut-) Vorbereitung geht es nicht.
- [ ] 11. Das Geschenk ist für ___ (die) Kollegin.
- [ ] 12. Sie spricht mit ___ (die) Lehrerin.
- [ ] 13. Ich habe von ___ (ein) Freund davon gehört.
- [ ] 14. Wir treffen uns um ___ (die) Ecke.
- [ ] 15. Gegenüber ___ (der) Bahnhof ist ein Café.

---

## Part 2: Tricky Questions (16–20)

- [ ] 16. Sie wohnt in ___ (eine) groß- Stadt. (location)
- [ ] 17. Er legt das Buch auf ___ (der) Tisch. (direction)
- [ ] 18. Ich erinnere mich ___ (an, die) Reise.
- [ ] 19. Mit ___ (kein) ___ (gut-) Deutsch ist die Prüfung schwer.
- [ ] 20. Sie nimmt ___ (an, der) Kurs teil. (an + dem = am)

---

---

## Answers (Antworten)

### Part 1: Regular Questions

**1. dem Bus** (mit + Dativ)  
- English: *mit* always Dativ. *der Bus* → *dem Bus*.  
- Deutsche Erklärung: *mit* + Dativ: **dem** Bus.

**2. dich** (auf + Akkusativ)  
- English: *warten auf* + Akkusativ. *du* → *dich*.  
- Deutsche Erklärung: *warten auf* + Akkusativ: **dich**.

**3. seiner … alten** (bei + Dativ)  
- English: *bei* + Dativ. Possessive feminine: *seiner*; adjective *-en*: *alten*.  
- Deutsche Erklärung: *bei* + Dativ: **seiner alten** Mutter.

**4. das Wochenende** (auf + Akkusativ)  
- English: *sich freuen auf* + Akkusativ. *das* stays *das* in Akk.  
- Deutsche Erklärung: *sich freuen auf* + Akkusativ: **das** Wochenende.

**5. der** (aus + Dativ)  
- English: *aus* always Dativ. *die Türkei* → *der Türkei*.  
- Deutsche Erklärung: *aus* + Dativ: **der** Türkei.

**6. der** (nach + Dativ)  
- English: *nach* + Dativ. *die Arbeit* → *der Arbeit*.  
- Deutsche Erklärung: *nach* + Dativ: **der** Arbeit.

**7. dich** (an + Akkusativ)  
- English: *denken an* + Akkusativ. *du* → *dich*.  
- Deutsche Erklärung: *denken an* + Akkusativ: **dich**.

**8. einem** (seit + Dativ)  
- English: *seit* + Dativ. *ein Jahr* (n) → *einem Jahr*.  
- Deutsche Erklärung: *seit* + Dativ: **einem** Jahr.

**9. dem** → **zum**  
- English: *zu* + *dem* = *zum*. Zu + Dativ.  
- Deutsche Erklärung: *zu* + *dem* = **zum** (zum Arzt).

**10. gute** (ohne + Akkusativ)  
- English: *ohne* + Akkusativ. *Vorbereitung* (f) → *die/gute*. No article: *ohne gute Vorbereitung* (or *ohne eine gute*). With "die" meaning "the": *ohne die gute Vorbereitung*. For "without good preparation": **ohne gute Vorbereitung** (no article, adjective -e for fem. Akk. no article).  
- Deutsche Erklärung: *ohne* + Akkusativ: **ohne gute** Vorbereitung (oder ohne eine gute Vorbereitung).

**11. die** (für + Akkusativ)  
- English: *für* + Akkusativ. *die Kollegin* stays *die* in Akk.  
- Deutsche Erklärung: *für* + Akkusativ: **die** Kollegin.

**12. der** (mit + Dativ)  
- English: *mit* + Dativ. *die Lehrerin* → *der Lehrerin*.  
- Deutsche Erklärung: *mit* + Dativ: **der** Lehrerin.

**13. einem** (von + Dativ)  
- English: *von* + Dativ. *ein Freund* → *einem Freund*.  
- Deutsche Erklärung: *von* + Dativ: **einem** Freund.

**14. die** (um + Akkusativ)  
- English: *um* + Akkusativ. *die Ecke* stays *die* in Akk.  
- Deutsche Erklärung: *um* + Akkusativ: **die** Ecke.

**15. dem** (gegenüber + Dativ)  
- English: *gegenüber* + Dativ. *der Bahnhof* → *dem Bahnhof*.  
- Deutsche Erklärung: *gegenüber* + Dativ: **dem** Bahnhof.

---

### Part 2: Tricky Questions

**16. einer großen** (in + Dativ, location)  
- English: *wohnen* = location → *in* + Dativ. Feminine: *einer*; adjective *-en*.  
- Deutsche Erklärung: Ort → *in* + Dativ: **einer großen** Stadt.

**17. den** (auf + Akkusativ, direction)  
- English: *legen* = direction (wohin?) → *auf* + Akkusativ. *der Tisch* → *den Tisch*.  
- Deutsche Erklärung: Wohin? → *auf* + Akkusativ: **den** Tisch.

**18. an die** (sich erinnern an + Akkusativ)  
- English: *sich erinnern an* + Akkusativ. *die Reise* stays *die* in Akk.  
- Deutsche Erklärung: *sich erinnern an* + Akkusativ: **an die** Reise.

**19. keinem … guten** (mit + Dativ)  
- English: *mit* + Dativ. *kein Deutsch* (n) → *keinem*; adjective *-en*: *guten*.  
- Deutsche Erklärung: *mit* + Dativ: **keinem guten** Deutsch.

**20. am** (teilnehmen an + Dativ: an dem = am)  
- English: *teilnehmen an* + Dativ. *an* + *dem* = *am*.  
- Deutsche Erklärung: *an* + *dem* = **am** Kurs.
