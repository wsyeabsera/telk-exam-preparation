# Two-Way Prepositions (Wechselpräpositionen)

**in, an, auf, über, unter, vor, hinter, neben, zwischen** can take either **Dativ** or **Akkusativ**, depending on meaning.

---

## Rule

| Question | Meaning       | Case      | Example (location vs direction)     |
|----------|---------------|-----------|--------------------------------------|
| **Wo?**   | Where? (location) | **Dativ**  | Ich wohne **in der** Stadt.          |
| **Wohin?**| Where to? (direction) | **Akkusativ** | Ich fahre **in die** Stadt.    |

**Deutsche Erklärung:** **Wo?** = Ort, keine Bewegung → Dativ. **Wohin?** = Richtung, Bewegung → Akkusativ.

---

## Examples

| Preposition | Wo? (Dativ)        | Wohin? (Akkusativ)     |
|-------------|--------------------|-------------------------|
| in          | in **der** Wohnung | in **die** Wohnung      |
| an          | an **der** Wand    | an **die** Wand         |
| auf         | auf **dem** Tisch  | auf **den** Tisch       |
| über        | über **der** Stadt | über **die** Brücke     |
| unter       | unter **dem** Tisch| unter **den** Tisch     |
| vor         | vor **dem** Haus   | vor **das** Haus        |
| hinter      | hinter **dem** Baum| hinter **den** Baum     |
| neben       | neben **der** Tür  | neben **die** Tür       |
| zwischen    | zwischen **den** Stühlen | zwischen **die** Stühle |

---

## Verbs That Indicate Location vs Direction

- **Location (Dativ):** wohnen, sein, stehen, liegen, hängen, sitzen  
- **Direction (Akkusativ):** gehen, fahren, legen, stellen, setzen, hängen (to hang something)

**Common mistake:** *in eine große Stadt wohnen* → **in einer großen Stadt wohnen** (wohnen = location = Dativ).
