# Prepositions – Exercises 02

**Instructions / Anweisungen:** Choose the correct case (Dativ or Akkusativ) and form. Answers at the bottom.

---

## Part 1: Regular Questions (1–15)

- [ ] 1. Das Buch liegt auf ___ (der) Tisch. (location)
- [ ] 2. Sie stellt die Vase auf ___ (der) Tisch. (direction)
- [ ] 3. Ich habe ___ (für, du) ein Geschenk.
- [ ] 4. Er arbeitet bei ___ (die) Firma.
- [ ] 5. Wir fahren durch ___ (der) Tunnel.
- [ ] 6. Sie kommt von ___ (eine) Reise zurück.
- [ ] 7. Ich gratuliere ___ (du) zum Geburtstag.
- [ ] 8. Ohne ___ (du) mache ich das nicht.
- [ ] 9. Seit ___ (der) Montag bin ich krank.
- [ ] 10. Der Hund sitzt unter ___ (der) Tisch. (location)
- [ ] 11. Sie geht in ___ (die) Schule. (direction: wohin?)
- [ ] 12. Er wohnt in ___ (die) Altstadt. (location)
- [ ] 13. Ich helfe ___ (sie, plural).
- [ ] 14. Wir treffen uns vor ___ (der) Kino.
- [ ] 15. Sie antwortet ___ (er) nicht.

---

## Part 2: Tricky Questions (16–20)

- [ ] 16. Das Bild hängt an ___ (die) Wand. (location: wo?)
- [ ] 17. Er hängt das Bild an ___ (die) Wand. (direction: wohin?)
- [ ] 18. Ich freue mich ___ (auf, ihr) Besuch.
- [ ] 19. Zwischen ___ (der) beiden Häusern steht ein Baum. (location)
- [ ] 20. Sie wartet ___ (auf, die) Antwort.

## Answers (Antworten)

### Part 1: Regular Questions

**1. dem** (auf + Dativ, location)  
- English: *liegen* = location (wo?). *auf* + Dativ. *der Tisch* → *dem Tisch*.  
- Deutsche Erklärung: Wo? → *auf* + Dativ: **dem** Tisch.

**2. den** (auf + Akkusativ, direction)  
- English: *stellen* = direction (wohin?). *auf* + Akkusativ. *der Tisch* → *den Tisch*.  
- Deutsche Erklärung: Wohin? → *auf* + Akkusativ: **den** Tisch.

**3. dich** (für + Akkusativ)  
- English: *für* + Akkusativ. *du* → *dich*.  
- Deutsche Erklärung: *für* + Akkusativ: **dich**.

**4. der** (bei + Dativ)  
- English: *bei* + Dativ. *die Firma* → *der Firma*.  
- Deutsche Erklärung: *bei* + Dativ: **der** Firma.

**5. den** (durch + Akkusativ)  
- English: *durch* always Akkusativ. *der Tunnel* → *den Tunnel*.  
- Deutsche Erklärung: *durch* + Akkusativ: **den** Tunnel.

**6. einer** (von + Dativ)  
- English: *von* + Dativ. *eine Reise* (f) → *einer Reise*.  
- Deutsche Erklärung: *von* + Dativ: **einer** Reise.

**7. dir** (gratulieren + Dativ)  
- English: *gratulieren* takes Dativ. *du* → *dir*.  
- Deutsche Erklärung: *gratulieren* + Dativ: **dir**.

**8. dich** (ohne + Akkusativ)  
- English: *ohne* + Akkusativ. *du* → *dich*.  
- Deutsche Erklärung: *ohne* + Akkusativ: **dich**.

**9. dem** (seit + Dativ)  
- English: *seit* + Dativ. *der Montag* → *dem Montag*.  
- Deutsche Erklärung: *seit* + Dativ: **dem** Montag.

**10. dem** (unter + Dativ, location)  
- English: *sitzen* = location. *unter* + Dativ. *der Tisch* → *dem Tisch*.  
- Deutsche Erklärung: Wo? → *unter* + Dativ: **dem** Tisch.

**11. die** (in + Akkusativ, direction)  
- English: *gehen* = direction (wohin?). *in* + Akkusativ. *die Schule* stays *die*.  
- Deutsche Erklärung: Wohin? → *in* + Akkusativ: **die** Schule.

**12. der** (in + Dativ, location)  
- English: *wohnen* = location. *in* + Dativ. *die Altstadt* → *der Altstadt*.  
- Deutsche Erklärung: Wo? → *in* + Dativ: **der** Altstadt.

**13. ihnen** (helfen + Dativ)  
- English: *helfen* + Dativ. *sie* (they) → *ihnen*.  
- Deutsche Erklärung: *helfen* + Dativ: **ihnen**.

**14. dem** (vor + Dativ or Akkusativ). Location "in front of the cinema" = *vor dem Kino*.  
- English: *treffen* = meeting at a place, location. *vor* + Dativ. *das Kino* → *dem Kino*.  
- Deutsche Erklärung: Ort → *vor* + Dativ: **dem** Kino.

**15. ihm** (antworten + Dativ)  
- English: *antworten* + Dativ. *er* → *ihm*.  
- Deutsche Erklärung: *antworten* + Dativ: **ihm**.

### Part 2: Tricky Questions

**16. der** (an + Dativ, location)  
- English: *hängen* (intransitive: the picture hangs) = location. *an* + Dativ. *die Wand* → *der Wand*.  
- Deutsche Erklärung: Wo? → *an* + Dativ: **der** Wand.

**17. die** (an + Akkusativ, direction)  
- English: *hängen* (transitive: to hang something) = direction. *an* + Akkusativ. *die Wand* stays *die*.  
- Deutsche Erklärung: Wohin? → *an* + Akkusativ: **die** Wand.

**18. auf ihren** (sich freuen auf + Akkusativ)  
- English: *sich freuen auf* + Akkusativ. *ihr Besuch* (m) in Akk: *ihren Besuch*. So: **auf ihren Besuch**.  
- Deutsche Erklärung: *sich freuen auf* + Akkusativ: **auf ihren** Besuch.

**19. den** (zwischen + Dativ, location)  
- English: *stehen* = location. *zwischen* + Dativ plural. *die Häuser* → *den Häusern*. So: **zwischen den beiden Häusern**.  
- Deutsche Erklärung: Wo? Plural: **zwischen den beiden Häusern** (Dativ: den + Häusern).

**20. auf die** (warten auf + Akkusativ)  
- English: *warten auf* + Akkusativ. *die Antwort* stays *die* in Akk.  
- Deutsche Erklärung: *warten auf* + Akkusativ: **auf die** Antwort.
