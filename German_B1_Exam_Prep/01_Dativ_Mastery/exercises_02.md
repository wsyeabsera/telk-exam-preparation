# Dativ Mastery – Exercises 02

**Instructions / Anweisungen:** Fill in the correct Dativ form. Focus on feminine, plural, and adjective endings. Answers are at the bottom.

---

## Part 1: Regular Questions (1–15)

- [ ] 1. Sie arbeitet bei ___ (der) Firma.
- [ ] 2. Ich warte auf ___ (du).
- [ ] 3. Er schenkt ___ (die) Frau Blumen.
- [ ] 4. Wir danken ___ (die) Lehrer für die Hilfe.
- [ ] 5. Das Handy gehört ___ (ich).
- [ ] 6. Seit ___ (ein) Woche bin ich krank.
- [ ] 7. Sie kommt aus ___ (die) Türkei.
- [ ] 8. Ich gehe zu ___ (der) Arzt. (zu + dem = zum)
- [ ] 9. Er erzählt ___ (wir) eine Geschichte.
- [ ] 10. Mit ___ (dein) Bruder habe ich gesprochen.
- [ ] 11. Sie hilft ___ (die) alt- Professorin.
- [ ] 12. Nach ___ (die) Arbeit fahre ich nach Hause.
- [ ] 13. Das gefällt ___ (sie, plural).
- [ ] 14. Von ___ (die) Reise bin ich müde.
- [ ] 15. Bei ___ (ein) gut- Freund wohne ich.

---

## Part 2: Tricky Questions (16–20)

- [ ] 16. In ___ (die) neu- Wohnung gibt es viel Licht. (location)
- [ ] 17. Ich habe ___ (die) nett- Kollegin geholfen.
- [ ] 18. Mit ___ (kein) Geld kann man nicht reisen.
- [ ] 19. Sie spricht mit ___ (die) jung- Studenten. (plural)
- [ ] 20. Von ___ (eine) lang- Fahrt war er erschöpft.

## Answers (Antworten)

### Part 1: Regular Questions

**1. der**  
- English: Feminine Dativ = *der*. Bei + Dativ.  
- Deutsche Erklärung: *die Firma* → Dativ: **der** Firma. *bei* + Dativ.

**2. dich** (Akkusativ!)  
- English: *warten auf* takes **Akkusativ**, not Dativ. So: *auf dich*.  
- Deutsche Erklärung: *warten auf* + **Akkusativ**: **dich**. Nicht Dativ!

**3. der**  
- English: Feminine Dativ = *der*. Schenken: *jemandem* (Dat) *etwas* (Akk).  
- Deutsche Erklärung: *die Frau* → Dativ: **der** Frau.

**4. dem** (if one male teacher) or **der** (if one female teacher). **den Lehrern** (plural).  
- English: If singular masculine: *der Lehrer* → *dem Lehrer*. If plural: *den Lehrern*.  
- Deutsche Erklärung: Singular maskulin: **dem** Lehrer. Plural: **den** Lehrern.

**5. mir**  
- English: Dativ pronoun for *ich* is *mir*. Gehören + Dativ.  
- Deutsche Erklärung: *ich* → Dativ: **mir**. *gehören* + Dativ.

**6. einer**  
- English: Feminine *eine Woche* in Dativ = *einer Woche*. Seit + Dativ.  
- Deutsche Erklärung: *eine Woche* → Dativ: **einer** Woche. *seit* + Dativ.

**7. der**  
- English: Feminine *die Türkei* in Dativ = *der Türkei*. Aus + Dativ.  
- Deutsche Erklärung: *die Türkei* → Dativ: **der** Türkei. *aus* + Dativ.

**8. dem** → **zum**  
- English: *zu* + *dem* = *zum*. Zu + Dativ.  
- Deutsche Erklärung: *zu* + *dem* = **zum** (zum Arzt).

**9. uns**  
- English: Dativ pronoun for *wir* is *uns*. Erzählen: *jemandem* (Dat) *etwas* (Akk).  
- Deutsche Erklärung: *wir* → Dativ: **uns**. *erzählen* + Dativ.

**10. deinem**  
- English: Possessive Dativ masculine = *deinem*. Mit + Dativ.  
- Deutsche Erklärung: *dein Bruder* → Dativ: **deinem** Bruder. *mit* + Dativ.

**11. der alten**  
- English: Feminine Dativ = *der*; adjective *-en*: *alten*.  
- Deutsche Erklärung: Dativ Femininum: **der** + **alten** Professorin.

**12. der**  
- English: Feminine *die Arbeit* in Dativ = *der Arbeit*. Nach + Dativ.  
- Deutsche Erklärung: *die Arbeit* → Dativ: **der** Arbeit. *nach* + Dativ.

**13. ihnen**  
- English: Dativ pronoun for *sie* (they) is *ihnen*.  
- Deutsche Erklärung: *sie* (Plural) → Dativ: **ihnen**. Kleingeschrieben (nicht *Sie* formell).

**14. der**  
- English: Feminine *die Reise* in Dativ = *der Reise*. Von + Dativ.  
- Deutsche Erklärung: *die Reise* → Dativ: **der** Reise. *von* + Dativ.

**15. einem guten**  
- English: Masculine indefinite Dativ = *einem*; adjective in Dativ = *-en*: *guten*.  
- Deutsche Erklärung: *ein Freund* → **einem**; Adjektiv Dativ: **guten** Freund.

### Part 2: Tricky Questions

**16. der neuen**  
- English: *In der Wohnung* = location (wo?) → Dativ. Feminine: *der*; adjective: *neuen*.  
- Deutsche Erklärung: *wohnen/sein* = Ort → *in* + Dativ: **der neuen** Wohnung.

**17. der netten**  
- English: Feminine Dativ = *der*; adjective always *-en* in Dativ: *netten*.  
- Deutsche Erklärung: Dativ Femininum: **der netten** Kollegin. Nicht *der nette*.

**18. keinem**  
- English: *Mit* + Dativ. *kein* in Dativ masculine/neuter = *keinem* (Geld is neuter).  
- Deutsche Erklärung: *mit* + Dativ. *kein Geld* (n) → **keinem** Geld.

**19. den jungen**  
- English: Plural Dativ = *den*; adjective *-en*; *Studenten* already ends in -n.  
- Deutsche Erklärung: Dativ Plural: **den jungen** Studenten.

**20. einer langen**  
- English: Feminine *eine Fahrt* in Dativ = *einer*; adjective: *langen*.  
- Deutsche Erklärung: *eine Fahrt* → Dativ: **einer**; Adjektiv: **langen** Fahrt.
