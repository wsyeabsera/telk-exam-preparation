# Vocabulary: Health and Body / Gesundheit und Körper

B1 vocabulary for health, body, and doctor visits.

---

## Body

| German      | English   | Example                        |
|-------------|-----------|---------------------------------|
| der Kopf    | head      | Mir tut der Kopf weh.           |
| das Auge    | eye       | Sie hat braune Augen.           |
| das Ohr     | ear       | Er hat Ohrenschmerzen.          |
| die Nase    | nose      | Die Nase ist zu.                |
| der Mund    | mouth     | Sie öffnet den Mund.            |
| der Hals    | throat    | Ich habe Halsschmerzen.         |
| die Schulter| shoulder  | Meine Schulter tut weh.         |
| der Arm     | arm       | Er hat den Arm gebrochen.       |
| die Hand    | hand      | Sie wäscht die Hände.          |
| der Finger  | finger    | Er hat sich am Finger geschnitten. |
| der Rücken  | back      | Ich habe Rückenschmerzen.       |
| der Bauch   | stomach   | Mir tut der Bauch weh.          |
| das Bein    | leg       | Sie hat sich das Bein verletzt.  |
| der Fuß     | foot      | Die Füße sind müde.             |
| das Herz    | heart     | Sein Herz schlägt schnell.     |
| die Lunge   | lung      | Rauchen schadet der Lunge.      |

## Health and Illness

| German        | English    | Example                          |
|---------------|------------|-----------------------------------|
| gesund        | healthy    | Sie ist gesund.                   |
| krank         | ill        | Er ist krank.                     |
| die Krankheit | illness    | Die Krankheit war schwer.         |
| die Grippe    | flu        | Sie hat die Grippe.                |
| die Erkältung | cold       | Ich habe eine Erkältung.           |
| Fieber        | fever      | Das Kind hat Fieber.              |
| Husten        | cough      | Er hat Husten.                    |
| Schnupfen     | runny nose | Sie hat Schnupfen.                |
| Schmerzen     | pain       | Ich habe Schmerzen.               |
| wehtun        | to hurt    | Mir tut der Kopf weh.             |
| sich erkälten | to catch a cold | Sie hat sich erkältet.      |
| der Arzt      | doctor     | Ich gehe zum Arzt.                |
| die Ärztin    | doctor (f) | Die Ärztin untersucht mich.      |
| die Praxis    | practice   | Die Praxis ist von 9 bis 12 Uhr geöffnet. |
| das Krankenhaus | hospital  | Er liegt im Krankenhaus.          |
| die Apotheke  | pharmacy   | In der Apotheke bekomme ich Medizin. |
| die Medizin   | medicine   | Ich nehme die Medizin.            |
| das Medikament| medication | Das Medikament hilft.             |
| die Tablette  | tablet     | Nehmen Sie eine Tablette.         |
| die Pille     | pill       | Sie nimmt die Pille.              |
| die Versicherung | insurance | Ich habe eine Krankenversicherung. |
| der Termin    | appointment | Ich habe einen Termin beim Arzt.  |
| untersuchen   | to examine | Der Arzt untersucht den Patienten. |
| verschreiben  | to prescribe | Der Arzt verschreibt ein Medikament. |
| operieren    | to operate | Sie wurde operiert.               |
| impfen       | to vaccinate | Die Kinder werden geimpft.      |
| sich ausruhen | to rest   | Sie muss sich ausruhen.           |
| sich erholen  | to recover | Er erholt sich gut.              |

---

## Phrases

- Ich bin krank und gehe zum Arzt.
- Mir tut der Kopf weh.
- Sie hat eine Erkältung.
- Der Arzt verschreibt ein Medikament.
- Ich habe einen Termin in der Praxis.
- In der Apotheke bekomme ich die Tabletten.
- Sie muss sich ausruhen.
