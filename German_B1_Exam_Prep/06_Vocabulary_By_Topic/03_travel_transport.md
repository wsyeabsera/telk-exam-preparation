# Vocabulary: Travel and Transport / Reisen und Verkehr

B1 vocabulary for travel, transport, and directions.

---

## Transport

| German       | English    | Example                          |
|--------------|------------|-----------------------------------|
| der Zug      | train      | Der Zug fährt um 8 Uhr.           |
| die Bahn     | train, railway | Ich fahre mit der Bahn.        |
| der Bus      | bus        | Der Bus kommt gleich.             |
| die U-Bahn   | metro      | Sie nimmt die U-Bahn.             |
| die Straßenbahn | tram    | Die Straßenbahn fährt ins Zentrum.|
| das Auto     | car        | Wir fahren mit dem Auto.          |
| das Fahrrad  | bicycle    | Er fährt mit dem Fahrrad.         |
| das Flugzeug | plane      | Das Flugzeug startet um 10 Uhr.   |
| das Schiff   | ship       | Sie fährt mit dem Schiff.         |
| das Motorrad | motorcycle | Er hat ein Motorrad.              |
| der Führerschein | driving licence | Sie hat den Führerschein.   |
| fahren       | to drive, go | Wir fahren in den Urlaub.        |
| fliegen      | to fly     | Sie fliegt nach Spanien.          |
| reisen       | to travel  | Er reist gern.                    |
| die Reise    | trip       | Die Reise war schön.              |
| die Fahrt    | journey    | Die Fahrt dauerte zwei Stunden.   |
| der Bahnhof  | station    | Wir treffen uns am Bahnhof.       |
| der Flughafen| airport    | Der Flughafen ist groß.           |
| der Hafen    | port       | Das Schiff ist im Hafen.           |
| die Haltestelle | stop     | An der nächsten Haltestelle steige ich aus. |
| die Fahrkarte | ticket    | Ich kaufe eine Fahrkarte.         |
| das Ticket   | ticket     | Hast du dein Ticket?              |
| die Reservierung | reservation | Ich habe eine Reservierung.    |
| einsteigen   | to get in  | Bitte einsteigen!                  |
| aussteigen   | to get out | Ich steige an der nächsten Station aus. |
| umsteigen    | to change  | Sie müssen in München umsteigen.   |
| die Abfahrt  | departure  | Die Abfahrt ist um 14 Uhr.        |
| die Ankunft  | arrival    | Die Ankunft war pünktlich.        |
| die Verspätung | delay    | Der Zug hat 20 Minuten Verspätung. |
| die Pünktlichkeit | punctuality | Pünktlichkeit ist wichtig.     |

## Travel and Holiday

| German        | English    | Example                           |
|---------------|------------|------------------------------------|
| der Urlaub    | holiday    | Im Urlaub fahren wir ans Meer.     |
| die Ferien    | holidays   | In den Ferien bin ich zu Hause.    |
| das Hotel     | hotel      | Wir übernachten im Hotel.         |
| die Übernachtung | overnight stay | Die Übernachtung war günstig. |
| das Gepäck    | luggage    | Das Gepäck ist schwer.             |
| der Koffer    | suitcase   | Ich packe meinen Koffer.            |
| die Reise     | journey    | Die Reise war anstrengend.         |
| der Pass      | passport   | Hast du deinen Pass dabei?         |
| das Visum     | visa       | Sie braucht ein Visum.              |
| die Grenze    | border     | Wir überqueren die Grenze.          |
| die Stadt     | city       | Die Stadt ist schön.               |
| das Land      | country    | In welchem Land wohnen Sie?       |
| die Landschaft| landscape  | Die Landschaft ist wunderschön.     |
| die Küste     | coast      | Sie wohnen an der Küste.           |
| der Strand    | beach      | Am Strand ist es schön.            |
| die Sehenswürdigkeit | sight   | Das ist eine Sehenswürdigkeit.     |
| besichtigen   | to visit (sights) | Wir besichtigen die Altstadt. |
| die Karte    | map        | Ich brauche eine Karte.            |
| die Richtung  | direction  | In welche Richtung geht es?        |
| links        | left       | Biegen Sie links ab.               |
| rechts       | right      | Die Bank ist rechts.                |
| geradeaus    | straight ahead | Gehen Sie geradeaus.           |
| die Kreuzung  | crossing   | An der Kreuzung links.             |

## Phrases

- Wir fahren mit dem Zug nach München.
- Sie hat eine Reservierung gemacht.
- Der Flug hat Verspätung.
- Ich steige an der nächsten Haltestelle aus.
- Im Urlaub waren wir am Strand.
- Können Sie mir sagen, wo der Bahnhof ist?
- Die Reise war sehr schön.
