# Vocabulary: Environment and Society / Umwelt und Gesellschaft

B1 vocabulary for environment, society, and current topics.

---

## Environment

| German        | English    | Example                            |
|---------------|------------|-------------------------------------|
| die Umwelt    | environment | Die Umwelt ist wichtig.             |
| die Natur     | nature     | Wir schützen die Natur.              |
| das Klima     | climate    | Das Klima verändert sich.            |
| die Umweltverschmutzung | pollution | Die Umweltverschmutzung ist ein Problem. |
| der Müll      | rubbish    | Wir trennen den Müll.                |
| recyclen      | to recycle | Sie recycelt Papier und Plastik.     |
| sparen        | to save    | Wir sparen Wasser.                   |
| die Energie   | energy     | Er spart Energie.                    |
| der Strom     | electricity | Der Strom kommt aus erneuerbaren Quellen. |
| das Wasser    | water      | Wasser ist kostbar.                  |
| die Luft      | air        | Die Luft ist sauber.                  |
| verschmutzen   | to pollute | Die Fabrik verschmutzt die Luft.     |
| der Wald      | forest     | Der Wald ist groß.                   |
| das Tier      | animal     | Die Tiere leben im Wald.             |
| die Pflanze   | plant      | Die Pflanzen brauchen Wasser.        |
| umweltfreundlich | eco-friendly | Das Auto ist umweltfreundlich.   |
| der Umweltschutz | environmental protection | Umweltschutz ist wichtig. |
| nachhaltig    | sustainable | Wir leben nachhaltiger.             |
| das Plastik   | plastic    | Weniger Plastik ist besser.         |
| die Verpackung | packaging  | Die Verpackung ist aus Papier.       |

## Society and Community

| German        | English    | Example                            |
|---------------|------------|-------------------------------------|
| die Gesellschaft | society   | Die Gesellschaft ändert sich.       |
| die Gemeinschaft | community | Wir leben in einer guten Gemeinschaft. |
| das Land      | country    | Deutschland ist ein schönes Land.  |
| die Stadt     | city       | In der Stadt wohnen viele Menschen.  |
| das Dorf      | village    | Sie wohnt in einem kleinen Dorf.    |
| die Nachbarschaft | neighbourhood | Die Nachbarschaft ist nett.   |
| der Staat     | state      | Der Staat unterstützt die Bildung.  |
| die Politik   | politics   | Er interessiert sich für Politik.   |
| die Regierung | government | Die Regierung hat eine neue Regel.  |
| das Gesetz    | law        | Das Gesetz verbietet das.           |
| die Regel     | rule       | Es gibt neue Regeln.                |
| die Pflicht   | duty       | Das ist meine Pflicht.               |
| das Recht     | right      | Jeder hat das Recht auf Bildung.     |
| die Freiheit  | freedom    | Die Freiheit ist wichtig.            |
| die Gleichheit | equality  | Wir wollen Gleichheit.               |
| die Kultur    | culture    | Die Kultur ist vielfältig.           |
| die Tradition | tradition  | Das ist eine alte Tradition.         |
| die Religion  | religion   | Sie spricht über Religion.           |
| die Migration | migration  | Migration ist ein Thema.             |
| der Migrant   | migrant    | Er ist Migrant.                      |
| die Integration | integration | Integration ist wichtig.          |
| die Arbeit    | work        | Arbeit gibt dem Leben Sinn.         |
| die Arbeitslosigkeit | unemployment | Die Arbeitslosigkeit sinkt.   |
| die Armut     | poverty    | Armut ist ein Problem.              |
| reich         | rich       | Sie ist reich.                       |
| arm           | poor       | Er ist arm.                          |
| helfen        | to help    | Wir helfen den Bedürftigen.          |
| spenden       | to donate  | Sie spendet Geld.                    |
| die Organisation | organisation | Die Organisation hilft.          |
| der Verein    | association | Er ist in einem Verein.            |
| freiwillig    | voluntary  | Sie arbeitet freiwillig.             |

## Media and Communication

| German       | English   | Example                          |
|--------------|-----------|-----------------------------------|
| die Nachrichten | news    | Ich sehe die Nachrichten.         |
| das Internet | internet  | Sie sucht im Internet.           |
| die Website  | website   | Die Website ist gut.              |
| die E-Mail   | email     | Schick mir eine E-Mail.          |
| die Soziale Medien | social media | Er nutzt soziale Medien.   |
| die Information | information | Ich brauche mehr Informationen. |
| die Meinung  | opinion   | Das ist meine Meinung.            |
| diskutieren  | to discuss | Wir diskutieren über das Thema.  |
| die Diskussion | discussion | Die Diskussion war interessant. |

## Phrases for Exam

- Die Umwelt ist wichtig für uns alle.
- Wir sollten weniger Müll produzieren.
- Ich trenne den Müll und spare Energie.
- In unserer Gesellschaft gibt es viele Themen.
- Jeder hat das Recht auf Bildung.
- Ich informiere mich im Internet.
- Das ist meine Meinung dazu.
