# Vocabulary: Work and Education / Arbeit und Bildung

B1-level vocabulary for work, school, and training.

---

## Work

| German         | English       | Example                              |
|----------------|---------------|---------------------------------------|
| die Arbeit     | work, job     | Meine Arbeit macht mir Spaß.          |
| arbeiten       | to work       | Er arbeitet bei einer Firma.          |
| der Beruf      | profession    | Was ist dein Beruf?                   |
| der Job        | job           | Sie sucht einen neuen Job.            |
| die Firma      | company       | Die Firma ist groß.                   |
| das Büro       | office        | Ich bin im Büro.                     |
| der Chef       | boss          | Der Chef ist zufrieden.               |
| die Chefin     | boss (f)      | Meine Chefin ist fair.                |
| der Kollege    | colleague     | Mein Kollege hilft mir.               |
| die Kollegin   | colleague (f) | Die Kollegin ist nett.                |
| der Mitarbeiter| employee      | Die Mitarbeiter haben eine Pause.     |
| die Mitarbeiterin | employee (f) | Sie ist Mitarbeiterin.            |
| der Kunde      | customer      | Der Kunde fragt nach dem Preis.       |
| die Kundin     | customer (f)  | Die Kundin ist zufrieden.             |
| verdienen      | to earn       | Sie verdient gut.                     |
| das Gehalt     | salary        | Das Gehalt wird am Monatsende gezahlt.|
| die Pause      | break         | In der Pause trinke ich Kaffee.       |
| kündigen       | to resign     | Er kündigt nächsten Monat.            |
| entlassen      | to dismiss    | Sie wurde entlassen.                   |
| die Stelle     | position, job | Es gibt eine freie Stelle.            |
| bewerben (sich)| to apply      | Ich bewerbe mich um die Stelle.       |
| die Bewerbung  | application   | Meine Bewerbung ist fertig.           |
| der Vertrag    | contract      | Der Vertrag läuft zwei Jahre.         |
| Vollzeit       | full-time     | Sie arbeitet in Vollzeit.             |
| Teilzeit       | part-time     | Er arbeitet in Teilzeit.              |
| die Ausbildung | training      | Er macht eine Ausbildung.            |
| der Praktikant | intern        | Der Praktikant lernt viel.            |
| die Erfahrung  | experience    | Ich habe viel Erfahrung.             |
| die Aufgabe    | task          | Das ist eine schwierige Aufgabe.       |
| das Meeting    | meeting       | Wir haben ein Meeting um 10 Uhr.      |
| die Besprechung| meeting       | Die Besprechung war lang.             |
| die E-Mail     | email         | Schick mir eine E-Mail.               |
| anrufen        | to call       | Ich rufe den Kunden an.               |
| die Konferenz  | conference    | Er ist auf einer Konferenz.           |

## Education

| German        | English     | Example                           |
|---------------|-------------|------------------------------------|
| die Schule    | school      | Sie geht zur Schule.                |

## Useful Phrases

- Ich arbeite bei einer Firma in Berlin.
- Sie hat sich um die Stelle beworben.
- Er studiert seit zwei Jahren.
- Die Prüfung war nicht einfach.
- Ich habe meine Ausbildung abgeschlossen.
- Wir haben ein Meeting um 14 Uhr.
- Sie verdient gut in ihrem Beruf.
