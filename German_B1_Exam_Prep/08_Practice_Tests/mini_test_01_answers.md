## Part 1 (1–15)

1. **der** (die Mutter → Dativ: der Mutter)  
2. **dir** (danken + Dativ)  
3. **ihren** (bei + Dativ plural: ihren Eltern)  
4. **den Kindern** (plural Dativ: den + Kindern)  
5. **mir** (gehören + Dativ)  
6. **der** (bei + Dativ, die Firma → der Firma)  
7. **dem** (mit + Dativ, der Bus → dem Bus)  
8. **einer langen** (von + Dativ, eine Reise → einer; adjective -en)  
9. **der** (nach + Dativ, die Arbeit → der Arbeit)  
10. **der netten** (Dativ feminine: der + netten)  
11. **einem** (ein Freund → Dativ: einem Freund)  
12. **ihnen** (gefallen + Dativ, sie plural → ihnen)  
13. **einem** (von + Dativ, ein Kollege → einem Kollegen)  
14. **uns** (helfen + Dativ)  
15. **der schwierigen** (Dativ feminine: der + schwierigen)
