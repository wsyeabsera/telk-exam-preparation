# Mini Test 02: Adjective Endings Focus

**Goal:** Practise adjective endings in Nominativ, Akkusativ, and Dativ.  
**Time:** About 20–25 minutes.  
**Answers:** See [mini_test_02_answers.md](mini_test_02_answers.md).

---

## Part 1 (Questions 1–15)

Fill in the correct adjective ending (and article/possessive if needed).

- [ ] 1. Ich helfe der ___ (nett-) Kollegin.
- [ ] 2. Er hat einen ___ (neu-) Job gefunden.
- [ ] 3. Das ist ein ___ (gut-) Buch.
- [ ] 4. Sie spricht mit dem ___ (alt-) Professor.
- [ ] 5. Bei der ___ (schwierig-) Aufgabe brauche ich Hilfe.
- [ ] 6. Den ___ (klein-) Kindern gefällt das Spiel.
- [ ] 7. Sie kauft eine ___ (neu-) Jacke.
- [ ] 8. Das ist der ___ (gut-) Freund von meiner Schwester.
- [ ] 9. Er kommt von einer ___ (lang-) Reise.
- [ ] 10. Die ___ (jung-) Frau arbeitet hier.
- [ ] 11. Nach dem ___ (gut-) Essen gehen wir spazieren.
- [ ] 12. Mit ___ (mein) ___ (alt-) Freund habe ich gesprochen.
- [ ] 13. Ich wohne in einer ___ (groß-) Stadt.
- [ ] 14. Das ___ (wichtig-) Thema ist die Umwelt.
- [ ] 15. Wir danken den ___ (freundlich-) Mitarbeitern.
